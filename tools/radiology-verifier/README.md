# Radiology Export Verifier

CLI for validating radiology export packs without invoking forensic tooling.

## Usage

```bash
node tools/radiology-verifier/verify-v1.0.cjs <pack-dir>
node tools/radiology-verifier/verify-v1.0.cjs <pack-dir> --json
```

See `docs/RADIOLOGY_EXPORT_VERIFICATION_CONTRACT.md` for the full contract.
